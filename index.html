Introduction:

Klondike, often referred to as solitaire, is a type of solitaire card game many people use to pass the time.
One of the main sources to play has been the version that shipped with older versions of Windows.
As this is no longer an option, this site will show another version windows has created to be played on their newer operating system as well as an online version.

Equipment/Materials:

If using a Windows machine: access to the Microsoft store.
If using a different machine: internet access.

Installation Instructions:

If you have a Windows machine: Go to the Microsoft store, search "Solitaire", and install it for free.
If you do not have a Windows machine use this link: https://www.solitr.com/

https://github.com/rossjc1996/P2

FAQs:

Q: How many cards do you flip when you need new ones? One or three?
A: The official rules state one, but three is also accepted for a more challenging experience.

Q: Are all games winnable?
A: No, actually. Statistics show that only 3/4th of games are winnable.

Q: Is the name of the game Klondike or Solitaire?
A: The official name is Klondike, but in the US and Canada it is referred to as "Solitaire", a shortened version of "a solitaire game". Solitaire refers to any tabletop game that can be played by oneself.
