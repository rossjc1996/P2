Introduction:

Klondike, often referred to as solitaire, is a type of solitaire card game many people use to pass the time.
One of the main sources to play has been the version that shipped with older versions of Windows.
As this is no longer an option, this site will show another version windows has created to be played on their newer operating system as well as an online version.


Equipment/Materials:

If using a Windows machine: access to the Microsoft store.
If using a different machine: internet access.


Installation Instructions:

If you have a Windows machine: Go to the Microsoft store, search "Solitaire", and install it for free.
If you do not have a Windows machine, click here.


How to play:

(Terminology can be referenced from picture below)

Step 1: Launching the game
(For Windows) Select Classic Solitaire (Klondike)
<
(For web) Follow the link provided above.

Step 2: Selecting Difficulty & Turn Number
(For Windows> Select your difficulty & turn number.
(For web) Click options to select turn number.
Tip: Turn number means the number of cards you flip from your stack. 1 makes the game easier.
Step 3: Objective
(From this step onward, both versions play the same)
The final goal is to have the four suits built aces through kings on the foundations.
Step 4: Progression
Set any available cards to their foundation, beginning with aces and going in order to kings.
Flip 1 (or 3) card(s) from your stock to your waste pile. (This is done automatically by clicking the stock)
Repeat this step until you all cards are in the foundation or there are no more cards in the stock.
Step 5: Resetting
Sometimes you'll run out of moves. That's okay! You can reset the stock and try again as many times as you like!
Click where the stock was and it will reset itself.
Repeat steps 4 and 5 until all cards are in the foundations or you run out of moves
Step 6: What if I get stuck?
That's okay if you cant get any further! Just push the "New Game" button found at the top of the screen for the browser. For Windows, push "Try Again"

FAQs:

Q: How many cards do you flip when you need new ones? One or three?
A: The official rules state one, but three is also accepted for a more challenging experience.

Q: Are all games winnable?
A: No, actually. Statistics show that only 3/4th of games are winnable.

Q: Is the name of the game Klondike or Solitaire?
A: The official name is Klondike, but in the US and Canada it is referred to as "Solitaire", a shortened version of "a solitaire game". Solitaire refers to any tabletop game that can be played by oneself.
